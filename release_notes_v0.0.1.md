# Qubetics Chain Abstraction Solver Node v0.0.1

This is the initial release (v0.0.1) of the **Solver Node** binary for the Qubetics Chain Abstraction network.

## What's Included
* **mpc-node**: The executable binary for Ubuntu 24.
* **mpc-node-mac**: The executable binary for macOS Tahoe 26.3.

## Getting Started

Please see the [README.md](https://github.com/Qubetics/solver-network-binary#readme) for complete instructions on:
- How to download the binary using `wget`
- Generating your Peer ID
- Onboarding and funding your node
- Understanding network requirements and rewards
